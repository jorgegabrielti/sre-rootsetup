# SRE ROOT SETUP
## SETUP to SRE start to work

### **Requisits**:
- Git
```bash
sudo apt install -y git
```

#### First, clone the repository:
```bash
git clone git@github.com:jorgegabrielti/sre-rootsetup.git
```

#### Then, execute **sre-setup** script:

set permission:
```bash
cd sre-rootsetup
chmod +x src/sre-setup
```
```bash
src/sre-setup
```
